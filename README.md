# Swiggy-SQL-Data-Analysis# Swiggy Food Delivery Data Analysis Using SQL

## 📌 Project Overview
This project focuses on analyzing a Swiggy food delivery dataset using MySQL to uncover insights related to restaurant performance, pricing trends, customer ratings, and delivery efficiency.

## 🎯 Objective
To analyze restaurant data and generate meaningful business insights using SQL. The project aims to identify top-performing restaurants, pricing patterns, and city-wise trends.

## 🛠️ Tools & Technologies
- MySQL
- MySQL Workbench
- Kaggle Dataset
- Canva

## 📊 SQL Concepts Applied
- SELECT, WHERE, ORDER BY
- GROUP BY & HAVING
- Aggregate Functions (COUNT, AVG, MAX, MIN)
- Window Functions (RANK, DENSE_RANK, ROW_NUMBER)
- JOINs (INNER, LEFT, RIGHT, SELF)
- Subqueries
- CASE Statements

## ❓ Key Analytical Questions
1. What are the top 3 restaurants in each city based on order price?
2. Which city has the highest number of restaurants?
3. Which city has the highest average pricing?
4. Which restaurants provide the fastest delivery service?
5. Which restaurants offer the best value for money?
6. How are restaurants distributed across different pricing categories?

## 📈 Key Insights
- Identified top-performing restaurants city-wise.
- Analyzed pricing trends across multiple cities.
- Classified restaurants into Budget, Mid-Range, and Premium categories.
- Evaluated delivery efficiency and restaurant ratings.
- Performed competitor analysis within the same city.

## ✅ Conclusion
This project demonstrates how SQL can be used to transform raw restaurant data into actionable business insights. By applying advanced SQL techniques, meaningful patterns related to restaurant performance, pricing, ratings, and delivery efficiency were identified.
